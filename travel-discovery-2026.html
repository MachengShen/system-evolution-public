confirmed

重庆

重点看山城层叠空间、江岸夜景、老街区、火锅/小面/江湖菜，以及武隆/乌江画廊 这类自然支线。城市内建议同时采样热门区和本地生活区，避免只被网红点牵引。

  • 城市样本：渝中半岛、白象街、十八梯、鹅岭/南山视角。
  • 自然样本：武隆天生三桥、仙女山、乌江画廊。
  • 餐饮问题：高分店是否模板化，差评是否集中在排队、卫生、服务或宰客。
confirmed

成都

重点看城市慢生活、川菜/小吃、茶馆、都江堰/青城山支线，以及去九寨沟前的补给 和节奏调整。成都可以作为“城市舒适度 + 西行门户”的双重样本。

  • 城市样本：老街巷、茶馆、小吃密集区、本地社区餐馆。
  • 支线样本：都江堰、青城山、熊猫谷与灌县古城生活场景。
  • 餐饮问题：名气、排队和真实好吃之间是否一致。
confirmed

九寨沟

重点看自然景观、沟外住宿、交通接驳、旺季人流和高海拔体力成本。官方通告强调 “沟内游、沟外住”，所以住宿候选应围绕景区外接驳与恢复质量。

  • 住宿样本：景区外，优先验证接驳、早餐、供暖/热水、安静度。
  • 体验样本：避开极端拥堵时段，记录体力、天气和拍照噪声。
  • 风险问题：平台照片和现场维护状态是否一致。

Local scout brief

本地向导小众野性路线候选

贵阳近郊校准

花溪镇山村、高坡、红枫湖周边、紫江地缝可作为第一轮轻量样本：离贵阳近， 能测试交通、徒步强度、乡野感、餐食补给和拍摄价值。

适合：48 小时内验证本地向导判断力。

贵州喀斯特深水样本

荔波茂兰、都匀螺蛳壳、独山夹缝岩、织金洞周边可以作为“野性 + 地貌 + 少商业化” 候选，但需要核验路况和安全边界。

适合：需要包车或本地向导的深度探索。

高山草甸与异质地貌

乌蒙大草原、韭菜坪、铜鼓荡等适合作为“开阔感/异星感/风场”的样本。优先验证 天气、露营合规、补给和撤退路线。

适合：本地向导偏好的野山/野地探索。

重庆外延自然线

武隆、乌江画廊、奉节天坑地缝、巫溪红池坝可作为重庆段的自然外延。先做交通 和住宿可信度，不急着全国追山。

适合：把山城城市体验延伸到峡谷/草甸。

Agent pipeline

信息采集与判断代理

Source discovery 从 OTA、地图、公开网页、Human 线索里发现候选点，不直接判断真假。
Authorized data 优先使用 Booking Demand API、Trip.com/携程合作入口、高德/腾讯/百度地图 API 等授权或公开接口。
Review forensics 分析刷评信号：集中爆发、模板语言、缺少细节、平台评分分裂、差评主题聚类、图文不一致。
Human scout prior 把本地向导和实地探店者的照片、价格、菜单、卫生、噪音、排队、交通、老板/服务互动转成高置信样本。
Ranker 只输出少量 shortlist：值得去、可试但有风险、评分高但疑似刷、只适合特定人群、不要去。

What to ask the local scout

回头问本地向导的低负担问题

本地向导熟悉当地、实地判断力强，应该让他补的是“现场判断”，不是让他读长文档。 最有价值的问题是：

  1. 你去贵阳/贵州时，哪些地方是真野、真舒服、不是平台包装出来的？
  2. 哪些小酒店/农庄/餐馆，评分看起来一般但现场很值得？
  3. 哪些高分点你觉得是刷出来的，为什么？
  4. 如果只让你带一个外地朋友走 48 小时，你会怎么排？
